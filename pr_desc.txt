## What
Added a visual loading spinner (`Loader2` from `lucide-react`) to the submit button on the login/registration form.

## Why
When users submit the form, the button text changes to "Signing in..." or "Creating account...", but without a visual indicator of movement, the interface can feel frozen during slow network requests. A spinner provides immediate, clear visual feedback that a background process is occurring.

## Before/After
Before: Button text updated, but no visual spinner was present.
After: Button displays the loading text alongside a spinning icon, vertically centered using `items-center`.

## Accessibility
Added `aria-hidden="true"` to the SVG spinner so screen readers do not announce the purely visual element, as the surrounding text already conveys the "Signing in..." state.
